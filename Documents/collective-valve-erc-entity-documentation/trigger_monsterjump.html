    trigger_monsterjump
    Half-Life trigger entity
    description

    The trigger_monsterjump entity allows you to define an area that will cause any monster walking into it to jump in the specified direction, height, and speed.

    properties

    The direction of the jump is set by the angle of the trigger_monsterjump entity.

    properties

  • height (integer) Jump Height (default: 128) - This is the height of the jump in units.
  • speed (integer) Jump Speed (default: 40) - This is the speed of the jump in units per second.
  • Master
  • angles (string) Pitch Yaw Roll (Y Z X) (default: 0 0 0)
