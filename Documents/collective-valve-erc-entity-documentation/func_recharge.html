    func_recharge
    Half-Life func entity
    description

    The func_recharge entity is a brush-based entity that acts as a HEV suit recharger.

    usage

    When using this entity in a deathmatch map, set the Deathmatch Recharge Delay (dmdelay) to the number of seconds it should take to recharge once it is emptied.
