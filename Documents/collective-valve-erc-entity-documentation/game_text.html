    game_text
    Half-Life game entity
    description

    The game_text allows you to display a text message on the screen.

    usage

    This entity is an alternate method of displaying text on the screen. The original method was to use an env_message entity, but the game_text is preferable. It is designed to make it easier to display messages that don't need to be translated into different languages (like the ones in Half-Life). Line breaks can be added to the text by using "\n" (e.g. "This is line one\nThis is line two")

      Note: Entering message text longer than 127 characters will cause problems, most likely ending in Worldcraft crashing. This issue will hopefully be resolved in the next release.
    X, y, effect, color, color2, fadein, fadeout, fxtime are identical to the same keys in titles.txt.

    properties

  • targetname (target_source) Name - This entity must be activated by another trigger.
  • target (target_destination) Target - Event to activate when game_text is triggered.
  • message (string) Message Text - This is the text that will be displayed.
  • color (color255) Color1 (default: 100 100 100)
  • color2 (color255) Color2 (default: 240 110 0)
  • fadein (string) Fade in Time (or character scan time) (default: 1.5) - This is the time it takes to fade each character.
  • fadeout (string) Fade Out Time (default: 0.5) - How long it takes to fade out the message after holding.
  • fxtime (string) Scan time (scan effect only) (default: 0.25) - This is the amount of time the highlight lags behind the leading edge of the text.
  • holdtime (string) Hold Time (default: 1.2) - How long to hold the message on the screen (after it scans out).
  • x (string) X (0 - 1.0 = left to right) (-1 centers) (default: -1)
  • y (string) Y (0 - 1.0 = top to bottom) (-1 centers) (default: -1)
  • channel (integer) Text Channel - The game_text can output to one of four channels, numbered 1 to 4. Triggering text output on the same channel as a previous game_text will clear the previous game_text output.

         1 - Channel 1 (default)
         2 - Channel 2
         3 - Channel 3
         4 - Channel 4

  • effect (integer) Text Effect

         0 - Fade In/Out (default)
         1 - Credits
         2 - Scan Out

  • Master

    • Flags
    This entity has the following flags. Flags can be either on or off. In the editor, these flags can be accessed in the Flags section of the entity properties.

  • 1 - All Players - send the message to all players. If this is not set, then the message goes only to the player who fires the entity.
