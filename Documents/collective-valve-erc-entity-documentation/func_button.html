    func_button
    Half-Life func entity
    description

    The func_button entity allows you to create a button (surprise surprise!).

    properties

  • targetname (target_source) Name - If a name is given to this entity, it can be activated by another trigger.
  • target (target_destination) Target - When the button is touched or used, this event will be activated.
  • globalname (string) Global Entity Name
  • _minlight (string) Minimum light level
  • delay (string) Delay before trigger (default: 0) - This is delay between when the button is used and the button's events are activated.
  • health (integer) Health (shootable if > 0) - If the button has to be shot rather than touched or used, specify a value here. When this amount in damage is reached, the button's targets will be activated.
  • lip (integer) Lip
  • Master - You can specify a multisource master here.
  • speed (integer) Speed (default: 5) - This is the speed the button will move from its non-activated to its activated state.
  • wait (integer) delay before reset (-1 stay) (default: 3) - This is delay between when the button is used and when it is once again usable.
  • locked_sentence (integer) Locked Sentence
  • locked_sound (integer) Locked Sound

         0 - None (default)
         2 - Access Denied
         8 - Small zap
         10 - Buzz
         11 - Buzz Off
         12 - Latch Locked

  • sounds (integer) Sounds

         0 - None (default)
         1 - Big zap & Warmup
         2 - Access Denied
         3 - Access Granted
         4 - Quick Combolock
         5 - Power Deadbolt 1
         6 - Power Deadbolt 2
         7 - Plunger
         8 - Small zap
         9 - Keycard Sound
         10 - Buzz
         11 - Buzz Off
         14 - Lightswitch

  • unlocked_sentence (integer) Unlocked Sentence

         0 - None (default)
         1 - Gen. Access Granted
         2 - Security Disengaged
         3 - Blast Door
         4 - Fire Door
         5 - Chemical Door
         6 - Radiation Door
         7 - Gen. Containment
         8 - Maintenance area

  • unlocked_sound (integer) Unlocked Sound

         0 - None (default)
         1 - Big zap & Warmup
         3 - Access Granted
         4 - Quick Combolock
         5 - Power Deadbolt 1
         6 - Power Deadbolt 2
         7 - Plunger
         8 - Small zap
         9 - Keycard Sound
         10 - Buzz
         13 - Latch Unlocked
         14 - Lightswitch

  • renderamt (integer) FX Amount (1 - 255) - see Rendering Properties
  • rendercolor (color255) FX Color (R G B) (default: 0 0 0) - see Rendering Properties
  • rendermode (integer) Render Mode - see Rendering Properties
  • renderfx (integer) Render FX - see Rendering Properties
  • angles (string) Pitch Yaw Roll (Y Z X) (default: 0 0 0)
  • light_origin (string) Light Origin Target
  • zhlt_lightflags (integer) ZHLT Lightflags

         0 - Default (default)
         1 - Embedded Fix
         2 - Opaque (blocks light)
         3 - Opaque + Embedded fix
         6 - Opaque + Concave Fix

    • Flags
    This entity has the following flags. Flags can be either on or off. In the editor, these flags can be accessed in the Flags section of the entity properties.

  • 1 - Don't move - If this is enabled, the button will instantly go from inactive to activated when it is touched or used without moving.
  • 32 - Toggle - If this flag is enabled, the button will pause in its active and inactive position, waiting to be touched or used again. Each time it is activated, it will activate its events.
  • 64 - Sparks - If this is enabled, the button will randomly emit sparks when it is in its inactive position.
  • 256 - Touch Activates - If this is enabled, it will be possible to activate the button simply by touching it rather than using the "use" key on it.
