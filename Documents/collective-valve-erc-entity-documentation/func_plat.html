    func_plat
    Half-Life func entity
    description

    The func_plat entity lets you create a platform that will move to its raised position when you walk onto it.

    usage

    The plat must be drawn in its extended (raised) position, but it will start at its normal (lowered) position in the level.

    properties

  • targetname (target_source) Name - If a name is given to this entity, it can be activated by another trigger.
  • height (integer) Travel altitude (can be negative) (default: 0) - This is the distance the object will move. do not set this to a negative value.
  • speed (integer) Speed (default: 50) - This value determines how fast the platform moves.

    • Flags
    This entity has the following flags. Flags can be either on or off. In the editor, these flags can be accessed in the Flags section of the entity properties.

  • 1 - Toggle - If this is enabled and the plat is triggerable, triggering it will toggle it between its lowered and raised position.
