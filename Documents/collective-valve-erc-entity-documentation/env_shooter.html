    env_shooter
    Half-Life env entity
    description

    The env_shooter entity allows you to to shoot a specific model or sprite.

    Unlike the gibshooter, which shares the majority of its properties, it is not used to spawn bloody gibs. Rather, you must specify the gib model in the env_shooter properties. More on this below.

    properties

  • targetname (target_source) Name - This entity must be triggered to become active. Give it a name, then activate it with a trigger.
  • scale (string) Gib Sprite Scale - When shooting a sprite, this value controls the scale of the sprite.
  • shootmodel (studio) Model or Sprite name - The value of this defines the model or sprite that will be shot from the env_shooter. The value must be in the format "models/modelname.mdl" or "sprites/spritename.spr".
  • skin (integer) Gib Skin (default: 0) - This value will produce different results for different models. Use an MDL viewer to view the different skins each model can have.
  • shootsounds (integer) Material Sound - This is the sound that will be heard when the gibs hit the ground. Valid sounds are:

         -1 - None (default)
         0 - Glass
         1 - Wood
         2 - Metal
         3 - Flesh
         4 - Concrete

  • delay (string) Delay between shots (default: 0) - This is the delay between when each gib will be expelled.
  • m_flGibLife (string) Gib Life (default: 4)
  • m_flVariance (string) Course Variance (default: 0.15)
  • m_flVelocity (integer) Gib Velocity (default: 200) - This is the speed at which the gib will be tossed.
  • m_iGibs (integer) Number of Gibs (default: 3) - This is the total number of gibs that will be spawned per activation.
  • renderamt (integer) FX Amount (1 - 255) - see Rendering Properties.
  • rendercolor (color255) FX Color (R G B) (default: 0 0 0) - see Rendering Properties.
  • rendermode (integer) Render Mode - see Rendering Properties.
  • renderfx (integer) Render FX - see Rendering Properties.
  • angles (string) Pitch Yaw Roll (Y Z X) (default: 0 0 0)

    • Flags
    This entity has the following flags. Flags can be either on or off. In the editor, these flags can be accessed in the Flags section of the entity properties.

  • 1 - Repeatable
